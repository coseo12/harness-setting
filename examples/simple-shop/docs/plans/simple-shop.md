# SimpleShop MVP 기획서

## 배경

AI 에이전트 프레임워크 간소화(15 -> 9 에이전트, 적응형 파이프라인) 검증을 위한 예제 프로젝트.
최소한의 복잡도로 프레임워크의 전체 파이프라인을 검증할 수 있는 쇼핑 앱을 구현한다.

## 목표

SimpleShop MVP: 제품 카탈로그 탐색 + 위시리스트 기능을 갖춘 웹 애플리케이션.

## 기능 요구사항

### 필수 기능

1. **제품 목록 그리드**: 전체 제품을 그리드 형태로 표시
2. **제품 검색**: 제품명 기반 검색
3. **제품 상세**: 제품 클릭 시 상세 정보 표시 (이름, 가격, 설명, 재고)
4. **위시리스트 추가/제거**: 하트 버튼으로 위시리스트에 추가/제거
5. **위시리스트 페이지**: 위시리스트에 담긴 제품 목록 조회

### 선택 기능

- 카테고리 필터: 카테고리별 제품 필터링
- 가격순 정렬: 가격 오름차순/내림차순 정렬

## 비기능 요구사항

- **반응형**: 모바일(480px) 1열, 데스크톱(1200px) 3열 그리드
- **접근성**: 시맨틱 HTML, ARIA 라벨, 키보드 접근성

## 화면 흐름

```
홈(제품 목록) → 제품 상세 → 위시리스트 페이지
```

## 범위 (Scope)

- 인증: 없음
- 데이터: 로컬 메모리 데이터 (DB 없음)
- 결제: 없음
