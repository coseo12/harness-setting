# 채팅 앱 기획서

## 개요
사용자가 로그인 후 채팅방을 생성/참여하여 실시간으로 메시지를 주고받는 웹 애플리케이션.

## 핵심 기능
1. **사용자 인증**: 이메일 기반 회원가입, 로그인, JWT 토큰
2. **채팅방 관리**: 방 생성, 목록 조회, 입장
3. **실시간 메시징**: WebSocket 기반 메시지 송수신
4. **접속자 관리**: 방별 온라인 사용자 표시
5. **메시지 이력**: 이전 대화 불러오기 (페이지네이션)

## 기술 스택
- Next.js 15, React 19, TypeScript
- SQLite + Prisma ORM
- Socket.IO (양방향 실시간 통신)
- jose + bcryptjs (인증)
- Vitest + Testing Library (테스트)
